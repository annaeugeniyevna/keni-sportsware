# Keni Sportsware

A responsive and modern website developed using **WordPrees** with a headless-inspired approach. The project focuses on clean design, performance optimization, and seamless user experience across both desctop and mobile devices.


## Key Features 

- **Custom Layouts**: Build with **Spectra** blocks to ensure flexible, high-performance container structure.
- **Responsive Design**: Fully adaptive interface optimized for various screen sizes.
- **Interactive Elements**: Custom CSS implementation for hover effects and animations to enhance user engagement.
- **Performance Focused**: Optimized image scaling and minimal reliance on heavy plugins to ensure fast loading speeds.

## Tech Stack

- **CMS**: WordPress
- **Editor/Builder**: Spectra (Gutenberg blocks)
- **Styling**: Custom CSS for tailored visual effects and responsive adjustments.
